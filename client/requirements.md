## Packages
recharts | For data visualization (line charts, bar charts, heatmaps)
framer-motion | For smooth page transitions and micro-interactions
lucide-react | For beautiful, consistent iconography
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind CSS classes without conflicts
date-fns | For date formatting and manipulation

## Notes
The application visualizes conversation analysis.
It needs a polling mechanism for analysis job status.
Uploads are handled via standard FormData POST request.
If the backend is unreachable, the frontend should gracefully handle errors or offer a demo mode.
